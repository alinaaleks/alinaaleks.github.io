### Вопрос

- Что писать в терминале, когда мне нужно отправить в GitHUB?

### Проблема / ситуация

Хотелось иметь быстрый доступ к используемым мною командам. Они забываются, особенно если не пользоваться несколько месяцев. Хотелось, чтобы был инструмент, чтобы быстро освежить память. Git-память.

### Цель / решение

Создать Telegram-бота-напоминалку.

### Результат / инсайты

- ускорена работа с Git
- снижено время поиска команд

### Инструменты

Python, Telegram API
